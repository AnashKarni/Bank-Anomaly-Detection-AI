# 🏦 Bank Anomaly Detection AI | Risk Intelligence Dashboard

Welcome to the **Bank Anomaly Detection AI**! This is a dynamic, interactive web dashboard built to analyze banking datasets and automatically flag anomalies or risks. 

This project is part of my **120-Day Data Science & AI Challenge**, geared towards building high-impact, real-world solutions.

### 🚀 Key Features
* **Data Ingestion:** Seamlessly loads and reads banking data (`banking.csv`).
* **Risk Intelligence:** Analyzes the data to highlight irregularities and potential anomalies.
* **Interactive UI:** Built a clean, user-friendly interface using Streamlit for instant insights.

### 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **Data Manipulation:** Pandas
